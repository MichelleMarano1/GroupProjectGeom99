Mapping Indigenous data across Canada

A web solution using Ersi Configurable apps to map Indigenous data that is severely underrepresented

How we did it

ArcGIS Pro

Found language data from Statistics Canada reviewed the data and fixed all of ontario since most of the languages were marked as "undetermined which is not the case. Made map in Arc pro then uploaded it right into Arc Online then finalized the web map to embed into Dashboards.

Dashboards

Figured out the data we wanted to download and manipulate in ArcGIS Pro was in a REST end point so we took the URL and made the map right in Arc Online. From there with our 3 URLS uploaded into the content, we used the spatial analysis tool to manipulate the data. From there we learned how to author a Dashboard from scratch and how to make the elements talk to each other. We added the necessary elements to a pie graph, charts and lists to communicate our data.

Experience Builder

To create our experience builder we added a variety of elements including the dashboards that were made previously. We added more elements in the form of widgets to better tell the story of Indigenous communities in Canada. The widgets we used included images, text, menus, and buttons.

Configurable Apps

To create a configurable app we used an existing My Scene Map and edited the data to display the Indigenous population and the European settlement in the years 1630 and 1740. A configurable app using "Compare" was used to create a Side by Side two scene map
